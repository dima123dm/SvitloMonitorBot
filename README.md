**Моніторинг світла** — це потужний асинхронний інструмент для автоматизованого моніторингу графіків відключень електроенергії. Бот забезпечує користувачів найактуальнішою інформацією про стан електромережі, використовуючи сучасні алгоритми збору та обробки даних.

---

## 🔥 Ключові можливості

### 👤 Для користувачів
* **Інтелектуальний моніторинг:** Система автоматично отримує та аналізує дані про графіки, забезпечуючи високу точність та оперативність інформації.
* **Розумні сповіщення:**
    * 🔦 **Pre-alert:** Попередження про відключення (таймер налаштовується: 5, 15, 30, 60 хв).
    * 💡 **Post-alert:** Сповіщення про відновлення світла (таймер налаштовується окремо).
    * ⚡ **Moment-alert:** Сповіщення в реальному часі — рівно в момент, коли світло повертається за графіком.
    * ⚠️ **Grey Zone:** Попередження про "можливі" відключення (сірі зони графіку).
    * 🔄 **Update-alert:** Миттєве повідомлення, якщо графік змінився протягом дня.
    * ☀️ **Ранкове зведення (06:00):** Щоденний графік на сьогодні (якщо заплановані відключення).
* **Персоналізація:**
    * Вибір режиму відображення: **"Чорний"** (коли світла немає) або **"Зелений"** (коли світло є).
    * Гнучке налаштування таймерів сповіщень під власні потреби.
* **Статистика:** Детальна аналітика відключень за останні 7 днів.
* **Підтримка:** Вбудована система тікетів для зручного зв'язку з адміністратором.

### 📢 Підтримка груп та каналів
* **Додавання бота в групи та канали:** Команда `/addtogroup` дає готові посилання для швидкого додавання бота у будь-який чат.
* **Deep Linking для каналів:** При додаванні бота в канал (де inline-кнопки не працюють), бот надсилає спеціальне посилання. Перейшовши за ним, адміністратор потрапляє в приватний чат із ботом і може налаштувати канал звідти.
* **Меню "Мої канали/групи":** В особистих налаштуваннях доступний список усіх підключених чатів із можливістю зручного керування кожним.
* **Ідентичні налаштування:** Меню групи повністю повторює особисті налаштування — типи сповіщень, таймери попереджень, режим відображення.
* **Актуальний графік:** Після першого налаштування групи/каналу бот одразу надсилає поточний графік у чат.

### 👨‍💻 Для адміністратора
* **Адмін-панель:** Повнофункціональне меню керування ботом прямо в Telegram.
* **Система підтримки:** Отримання та відповідь на звернення користувачів (тікети) через інтерфейс бота.
* **Розсилка:** Інструмент для масової відправки важливих повідомлень усім користувачам.
* **Автоматичні бекапи:** Щоденна автоматична відправка резервної копії бази даних адміністратору в особисті повідомлення (о 03:00).

## 📂 Структура проєкту

* 📄 **`main.py`**: Точка входу. Ініціалізація бота, бази даних та запуск планувальника завдань.
* ⚙️ **`config.py`**: Конфігураційні змінні та налаштування середовища.
* 🤖 **`handlers.py`**: Логіка інтерфейсу користувача. Обробка команд, навігація по меню, адмін-панель, система тікетів та управління групами/каналами.
* 🧠 **`scheduler.py`**: "Серце" бота. Відповідає за:
    * Періодичну перевірку та оновлення даних.
    * Розумну систему розсилки в особисті та в групи/канали (фільтрація черг для уникнення спаму).
    * Відправку сповіщень за таймерами.
    * Ранкове зведення о 06:00.
    * Створення та відправку автоматичних бекапів.
* 🌐 **`api_utils.py`**: Модуль роботи з даними. Містить алгоритми запитів, обробки інформації та формування часових інтервалів.
* 🗄 **`database.py`**: Шар взаємодії з даними. Зберігання профілів користувачів, підписок груп/каналів, статистики, тікетів та системних налаштувань.

## 🚀 Встановлення та запуск

### Крок 1. Клонування репозиторію
```bash
git clone [https://github.com/mriamys/SvitloMonitorBot.git](https://github.com/mriamys/SvitloMonitorBot.git)
cd SvitloMonitorBot
```
### Крок 2. Налаштування оточення
```bash

python3 -m venv venv
source venv/bin/activate  # Для Linux/macOS
# venv\Scripts\activate   # Для Windows
```
### Встановіть залежності:
```bash
pip install -r requirements.txt
```
Крок 4. Запуск
```bash
python3 main.py
```
